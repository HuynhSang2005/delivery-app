# Execution Checklist

Checklist dùng nhanh cho dev + AI-agent khi chạy task trong `docs/plan`.

## A. Intake

- [ ] Đã xác định scope folder đúng
- [ ] Đã mở AGENTS đúng layer
- [ ] Đã đọc docs refs bắt buộc

## B. Claim

- [ ] Đã có issue Beads map 1:1 với mark-task
- [ ] Đã claim issue
- [ ] Đã xác định verification mode
- [ ] Đã ghi touched paths expected

## C. Execute

- [ ] Làm đúng scope task, không mở rộng âm thầm
- [ ] Nếu có blocker mới, tạo issue mới (discovered-from)
- [ ] Nếu đổi acceptance/scope, cập nhật docs trước

## D. Verify

- [ ] Có verification commands/checks
- [ ] Có test note (`n/a` nếu docs-only)
- [ ] Có kết quả pass/fail rõ ràng

## E. Close

- [ ] touched paths actual đã ghi
- [ ] drift check đã ghi
- [ ] follow-up issue đã tạo nếu cần
- [ ] issue đã close với lý do rõ ràng
